# Глоссарий

> 📚 **Контекст:** этот словарь помогает читать [Product Vision](../docs/vision.md),
> [Product Concept](../docs/product-concept.md) и остальные
> [гайды](README.md) на одном языке.

> 📌 **Статус:** Draft — черновик для согласования.

## Для кого этот документ

Для любого пользователя Хаба, который встретил незнакомый термин и хочет быстро
понять его смысл человеческим языком, без технических подробностей.

## Сценарий

Документ нужен, когда при чтении гайдов, Vision или Concept встретилось слово,
значение которого хочется уточнить.

## Пошаговая инструкция

1. Найдите термин в таблице ниже (термины отсортированы по алфавиту).
2. Прочитайте короткое определение простыми словами.
3. При необходимости перейдите по ссылке на связанный гайд или документ.

## Что делать, если...

- **Термина нет в списке** → загляните в [Product Concept](../docs/product-concept.md)
  или задайте вопрос команде; при повторении термин стоит добавить сюда.
- **Определение кажется слишком общим** → перейдите по ссылке на профильный
  гайд за подробностями.
- **Встретили технический термин из глубинных документов** → этот словарь
  намеренно описывает понятия для пользователей; технические термины живут в
  технической документации Хаба.
