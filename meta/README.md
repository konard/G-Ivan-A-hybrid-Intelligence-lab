# Meta

Каталог для служебных артефактов развития самого репозитория.

## Что здесь хранить

| Артефакт | Назначение |
| --- | --- |
| Artifact map | Реестр ключевых документов, владельцев, статусов и связей. |
| Maturity model | Статусы `draft`, `reviewed`, `published`, `superseded`. |
| Repository backlog | Ссылки на follow-up issues, которые меняют структуру или правила. |
| Navigation notes | Карты переходов между frameworks, education, research и projects. |

Пока каталог содержит только README. Детальный artifact map стоит добавлять,
когда появится достаточно документов, чтобы ручная навигация стала дорогой.
