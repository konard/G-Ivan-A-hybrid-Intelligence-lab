# hybrid-Intelligence-lab

`hybrid-Intelligence-lab` - центральный knowledge hub для исследований,
образовательных материалов, методологий и проектных артефактов по
управляемой работе гибридных human + AI команд.

Репозиторий не является production-кодовой базой. Production-проекты могут
жить в отдельных spoke-репозиториях и ссылаться сюда как на источник
методологии, исследований, образовательных программ и правил работы с ИИ.
