# Структура курса "Практика применения промпт-инжиниринга в работе БА"

## Целевая аудитория
Бизнес-аналитики компании Манго Телеком с начальным уровнем знаний в области ИИ

## Цели курса
1. Обозначить основные возможности применения ИИ и ИИ-агентов в создании документации БА
2. Синхронизировать знания команды в предметной области
3. Презентовать промпты для обработки результатов транскрибации встреч и валидации/генерации функциональных требований
4. Рассмотреть подходы в подготовке ИИ-модели к работе с промптом

## Структура курса

### Модуль 1: Введение в ИИ для бизнес-аналитиков (теоретическая часть)
**Продолжительность:** 30-40 минут

**Темы:**
1. Что такое промпт-инжиниринг и почему это важно для БА
2. Основные возможности ИИ в работе БА
3. Типы задач БА, которые можно автоматизировать с помощью ИИ
4. Ограничения и этические аспекты использования ИИ

### Модуль 2: Основы промпт-инжиниринга (практическая часть)
**Продолжительность:** 40-50 минут

**Темы:**
1. Структура эффективного промпта
2. Техники промпт-инжиниринга (Chain-of-Thought, Step-back, etc.)
3. Best practices для БА в ИТ-телекоме
4. Простые примеры промптов для повседневных задач БА

### Модуль 3: Работа с транскрипциями встреч (практическая часть)
**Продолжительность:** 50-60 минут

**Темы:**
1. Проблемы ASR-систем и их решение
2. Обзор промптов BA_ASR (v_2_express, v_6_4, v_7)
3. Практическая работа: очистка и постобработка транскрипции
4. Извлечение требований из транскрипций

### Модуль 4: Генерация и валидация функциональных требований (практическая часть)
**Продолжительность:** 50-60 минут

**Темы:**
1. Стандарты документирования в Манго Телеком
2. Обзор промптов BA_FD (gen_v_4_2, gen_v_4_3, valid_v3_0)
3. Практическая работа: генерация ФТ по шаблону
4. Валидация требований с использованием промптов

### Модуль 5: Подготовка ИИ-модели к работе (продвинутый уровень)
**Продолжительность:** 40-50 минут

**Темы:**
1. Контекстуализация: передача знаний модели
2. Использование примеров и шаблонов
3. Итеративное улучшение промптов
4. Создание библиотеки промптов для команды

### Модуль 6: Интеграция ИИ в рабочий процесс БА (заключительная часть)
**Продолжительность:** 30-40 минут

**Темы:**
1. Выбор ИИ-платформы для работы
2. Организация работы с промптами в команде
3. Метрики эффективности использования ИИ
4. Roadmap развития навыков промпт-инжиниринга

## Общая продолжительность курса
3.5-4.5 часа (с перерывами: 4-5 часов)

## Формат обучения
- Презентация с иллюстрациями (10-15 слайдов как основа)
- Практические демонстрации на реальных примерах
- Интерактивные упражнения
- Раздаточные материалы со ссылками на промпты

## Материалы курса
1. Презентация (10-15 слайдов)
2. Сценарий для преподавателя (развернутый текст лекции)
3. Доступ к wiki с промптами:
   - BA_ASR_v_2_express
   - BA_ASR_v_6_4
   - BA_ASR_v_7
   - BA_FD_gen_v_4_2
   - BA_FD_gen_v_4_3
   - BA_FD_valid_v3_0
4. Практические примеры и кейсы
5. Чек-листы и шаблоны для работы

## Ожидаемые результаты
После прохождения курса участники смогут:
1. Понимать основные принципы работы с ИИ в контексте задач БА
2. Составлять эффективные промпты для своих задач
3. Использовать готовые промпты для обработки транскрипций
4. Применять промпты для генерации и валидации ФТ
5. Интегрировать ИИ-инструменты в свой рабочий процесс
