# Research: Reputation Technologies (GRA)

Исследования **репутационных технологий** и фреймворка **GRA — Gravitational & Magnetic
Reputation Architecture** (публичный слоган *Global Reputation Agreement*). Материалы извлекают
видение фаундера (Иван Гулиенко) из приложенных к issue
[#260](https://github.com/G-Ivan-A/hybrid-Intelligence-lab/issues/260) данных, формализуют его как
фреймворк и проект **международного стандарта**, сравнивают с 15 мировыми проектами и проектируют
**отдельный репозиторий** в экосистеме Hub. Это **исследовательский домен** (`scope: repo-wide`):
здесь только отчёты — артефакты фреймворка (код, отдельное репо) рекомендуются, но не создаются
(ограничение issue #260).

Логическое продолжение раздела «репутационные технологии» из
[../hub/2026-06-20-ecosystem-architecture-research.md](../hub/2026-06-20-ecosystem-architecture-research.md),
который зафиксировал отсутствие сырых данных по репутации; issue #260 их приложил, а этот домен —
анализирует.
