# Проект: Идеи для курса БА по промпт-инжинирингу

Версия: 0.1

Дата: 2026-05-26

`projects/education-ba-prompt/` - песочница для накопления идей, обсуждений и
практик, которые позже можно будет структурировать в обучающий курс для
бизнес-аналитиков. На этой стадии важнее фиксировать полезные наблюдения,
рабочие шаблоны и вопросы, чем преждевременно закреплять финальную программу.

## Scope: Sandbox

Raw ideas area. Not a public project until owner + roadmap are assigned.

Граница песочницы: здесь можно накапливать наблюдения, черновики упражнений и
практические примеры, но эти материалы не являются опубликованной программой,
обязательным curriculum-артефактом или active project area до назначения owner и
roadmap.

## 🎯 Назначение

Накопление идей, обсуждений и практик для будущего обучающего курса для
бизнес-аналитиков.

## 📋 Что фиксируем

- Термины и концепции (output stability, consistency, controlled generation,
  structured output).
- Практические кейсы из работы БА (ТЗ, валидация, интеграции, коммуникация с
  заказчиком).
- Шаблоны промптов, которые работают.
- Идеи модулей, упражнений, форматов (чат, воркшоп, самообучение).
- Наблюдения о типичных ошибках, критериях качества и ограничениях применения
  ИИ в аналитической работе.

## 🔄 Процесс

1. Зафиксируй идею в `course-ideas.md` или создай отдельный файл в `docs/`.
2. Обсуди в комментарии / чате.
3. При согласовании - переноси в структуру курса позже.

Рабочее правило: сначала сохраняем практический контекст и примеры, затем
выделяем повторяющиеся паттерны. Если идея не готова к модулю, она остается в
песочнице с вопросами для проверки.
