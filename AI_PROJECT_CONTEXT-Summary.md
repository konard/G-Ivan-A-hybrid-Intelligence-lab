# Контекст проекта hybrid-Intelligence-lab (Summary)

> 🧭 **Уровень детализации: Summary.** Это сжатый контекст для оценки задач и
> внешних источников без чтения всего репозитория. Полный «как» — в
> [docs/ecosystem-map.md](docs/ecosystem-map.md), [CONCEPT.md](CONCEPT.md) и
> [docs/vision.md](docs/vision.md). Точка входа по экосистеме —
> [docs/ecosystem-map-Index.md](docs/ecosystem-map-Index.md).

- **Цель:** governance-first knowledge hub экосистемы — методологический центр,
  который хранит общие правила, стандарты, шаблоны и практики гибридной
  human + AI работы по модели hub-and-spoke.
- **Текущий фокус:** Sprint 3 «Hybrid Minimum Bootstrap» — система внешних знаний
  (реестр + инсайты), рамка уровней детализации `Index → Summary → Full` и
  экосистемная карта проектов.
- **Активные стандарты:** [standards/file-naming.md](standards/file-naming.md),
  [standards/executable-documentation-standard.md](standards/executable-documentation-standard.md),
  [standards/frontmatter-standard.md](standards/frontmatter-standard.md),
  [governance/rfc/knowledge-lifecycle-proposal.md](governance/rfc/knowledge-lifecycle-proposal.md),
  [governance/repo-model.md](governance/repo-model.md) (Anti-Inflation).
- **Границы (НЕ делаем сейчас):** Хаб — не production-кодовая база (код живёт в
  споках); не вводим CI-скраперы и авто-парсинг документов; не плодим каталоги и
  документы «на вырост» без доказанной операционной боли (Anti-Inflation).
- **Горизонты / Актуально для будущих фаз:** Материалы по оркестрации и
  мультиагентным системам накапливать для следующих фаз развития агентной
  работы и обучающих курсов. Исследования по графовой структуре связей,
  репутационному инжинирингу и зрелости внешних источников сохранять как задел
  для будущей эволюции базы знаний.
